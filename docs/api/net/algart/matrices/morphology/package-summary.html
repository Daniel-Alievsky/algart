Package net.algart.matrices.morphology

Algorithms of mathematical morphology and rank operations for n-dimensional AlgART matrices.

See:
          Description

Interface Summary
Interface Description
CustomRankPrecision Complete description of precision characteristics of rank operations, described in RankMorphology interface.
Morphology Mathematical morphology over n-dimensional matrices with a random n-dimensional structuring element (shape), represented by Pattern class.
RankMorphology Rank operations over n-dimensional matrices: percentile, rank, mean between given percentiles or values, etc., calculated on all matrix elements in an aperture with the fixed shape, represented by Pattern class.
 

Class Summary
Class Description
AbstractMorphology A skeletal implementation of the Morphology interface to minimize the effort required to implement this interface.
AbstractRankMorphology A skeletal implementation of the RankMorphology interface to minimize the effort required to implement this interface.
BasicMorphology The simplest complete implementation of Morphology interface.
BasicRankMorphology Almost complete implementation of RankMorphology interface with an instantiation method of some complete implementation.
ContinuedMorphology The filter allowing to transform any Morphology object to another instance of that interface, which uses some non-trivial form of continuation outside the source matrix.
ContinuedRankMorphology The filter allowing to transform any RankMorphology object to another instance of that interface, which uses some non-trivial form of continuation outside the source matrix.
IterativeErosion Iterative erosion of the matrix.
IterativeOpening Iterative opening of the matrix.
RankProcessors A set of streaming aperture processors, implementing all operations, specified in RankMorphology interface.
TiledMorphology The filter allowing to transform any Morphology object to another instance of that interface, which uses some given tiler for processing the source matrix (an argument of Morphology methods).
TiledRankMorphology The filter allowing to transform any RankMorphology object to another instance of that interface, which uses some given tiler for processing the source matrices (arguments of RankMorphology methods).
 

Enum Summary
Enum Description
Morphology.SubtractionMode Subtraction mode for some methods of Morphology interface.
RankPrecision Some predefined precision characteristics of rank operations, described in RankMorphology interface.
