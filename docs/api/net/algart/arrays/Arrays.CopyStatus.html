public boolean strictMode()
Returns true if and only if the array was copied in "strict" mode, i.e. by Arrays.copy(ArrayContext, UpdatableArray, Array, int, boolean) or Matrices.copy(ArrayContext, Matrix, Matrix, int, boolean) method with the argument strictMode=true or by any other method of copying arrays. Note: if this value is false, it does not guarantee that some "non-strict" algorithm was really used, it does only mean that the corresponding strictMode argument of the copying method was false.

Returns:
true the array was copyied strictly, false if it was copyied by Arrays.copy(ArrayContext, UpdatableArray, Array, int, boolean) or Matrices.copy(ArrayContext, Matrix, Matrix, int, boolean) method with strictMode=false.
