popDouble

double popDouble()
Removes the element at the top of this stack and returns its value, or throws EmptyStackException if the stack is empty.

It this object is an AlgART double array, implementing MutableDoubleArray interface, and it is not empty, the same action may be performed by the following code:

 double result = doubleArray.getDouble(doubleArray.length()-1);
 doubleArray.length(doubleArray.length()-1);

Returns:
the element at the top of this stack (it is removed from the stack by this method).
Throws:
java.util.EmptyStackException - if this stack is empty.

pushDouble

void pushDouble(double value)
Appends value element to the end of this stack.

It this object is an AlgART double array, implementing MutableDoubleArray interface, the same action may be performed by the following code:

 doubleArray.length(doubleArray.length()+1);
 doubleArray.setDouble(doubleArray.length()-1, value);

Parameters:
value - to be added to the top of this stack.
Throws:
TooLargeArrayException - if the resulting stack length is too large for this type of stacks.
